# Howrse Manager

**Howrse Manager** — каркас Tampermonkey-приложения для Ловади / Howrse.

Версия `0.1.0` закладывает основу будущего менеджера табуна: интерфейс, темы, лог, состояние приложения, настройки и заготовку адаптера для `www.lowadi.com`.

## Что есть в v0.1

- плавающее окно приложения на `www.lowadi.com`;
- современный интерфейс со страницами;
- светлая, тёмная и автоматическая тема;
- сохранение настроек в браузере;
- кнопки `Старт`, `Пауза`, `Стоп`;
- базовый лог событий;
- раздел разработчика;
- заготовка архитектуры:
  - `Application`;
  - `EventBus`;
  - `Storage`;
  - `Logger`;
  - `StateManager`;
  - `SettingsManager`;
  - `DelayManager`;
  - `RouteManager`;
  - `LowadiAdapter`;
  - `UIManager`.

## Чего пока нет

В v0.1 скрипт **не выполняет игровые действия**:

- не кликает по кнопкам;
- не прогоняет табун;
- не тренирует лошадей;
- не записывает в КСК.

Эта версия нужна как безопасный фундамент для следующих этапов.

## Установка для теста

1. Установить расширение Tampermonkey.
2. Открыть файл [`howrse-manager.user.js`](./howrse-manager.user.js).
3. Скопировать содержимое файла в новый userscript Tampermonkey.
4. Сохранить скрипт.
5. Открыть `https://www.lowadi.com/`.

На странице должна появиться кнопка/панель **Howrse Manager**.
