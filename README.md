# Howrse Manager

**Howrse Manager** — Tampermonkey-приложение для Ловади / Howrse.

Проект создаётся как будущий умный менеджер табуна: с красивым интерфейсом, модульной архитектурой, настройками, логом, анализом лошади и постепенным добавлением автоматизации.

## Текущая версия

`v0.3.1` — гибридный табунный режим с правками интерфейса.

## Что есть сейчас

- плавающее окно приложения на `www.lowadi.com`;
- современный интерфейс со страницами;
- светлая, тёмная и автоматическая тема;
- сохранение настроек в браузере;
- кнопки `Старт`, `Пауза`, `Продолжить`, `Стоп`;
- базовый лог событий;
- раздел разработчика;
- определение типа страницы;
- первичный анализ текущей лошади:
  - имя;
  - энергия;
  - здоровье;
  - настроение;
  - возраст;
  - пол;
  - наличие кнопки следующей лошади;
- гибридный табунный режим:
  - текущая лошадь;
  - анализ;
  - запоминание обработанной лошади;
  - переход к следующей лошади;
  - продолжение после загрузки новой страницы;
  - остановка при повторе лошади;
- режим лимита прогона:
  - ручной лимит;
  - `Авто` — до конца завода;
- запоминание открытого раздела после обновления страницы;
- живое обновление времени работы;
- исправленная прокрутка внутри окна приложения.

## Чего пока нет

Скрипт пока **не выполняет уход и тренировки**:

- не чистит;
- не кормит;
- не поит;
- не укладывает спать;
- не тренирует;
- не записывает в КСК.

Эти действия будут добавляться в следующих версиях.

## Установка для теста

1. Установить расширение Tampermonkey.
2. Открыть файл [`howrse-manager.user.js`](./howrse-manager.user.js).
3. Скопировать содержимое файла в новый userscript Tampermonkey.
4. Сохранить скрипт.
5. Открыть `https://www.lowadi.com/`.

## Как тестировать v0.3.1

1. Обновить код скрипта в Tampermonkey.
2. Открыть страницу конкретной лошади.
3. Проверить прокрутку внутри окна Howrse Manager.
4. Открыть раздел `Прогон`.
5. Проверить режим `Максимум лошадей за запуск`:
   - `Ручной лимит`;
   - `Авто — до конца завода`.
6. Открыть любой раздел, обновить страницу и проверить, что этот раздел открылся снова.
7. Запустить прогон и проверить, что время работы обновляется само.
