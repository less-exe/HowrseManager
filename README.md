# 🐴 Howrse Manager

Умный менеджер-ассистент для браузерной игры **Ловади** (Lowadi/Howrse).
Запускается как пользовательский скрипт через **Tampermonkey**.

> **Текущая версия: 0.1.0 — MVP «Глаза» 👁️**
> Приложение учится *правильно видеть* данные лошади. Действия (кормление, уход,
> прогон табуна) добавляются постепенно на следующих версиях — аккуратно и по одному.

---

## ✨ Возможности версии 0.1

- 🎨 Красивый современный интерфейс (тёмная/светлая тема, перетаскивание окна)
- 🧭 Меню из 10 разделов (готовые и «скоро»)
- 👁️ **Режим «Глаза»** — чтение параметров лошади: имя, энергия, здоровье,
  настроение, возраст, пол, корм, миссия
- 📋 Живой лог всех действий
- 📊 Статистика сессии
- ⚙️ Настройки с сохранением
- 🧑‍💻 **Раздел «Разработчик»** — диагностика: что скрипт нашёл на странице,
  а что нет (с кнопкой «Скопировать отчёт»)

> ⚠️ В этой версии приложение **только читает** данные и **ничего не нажимает**
> в игре. Это безопасный фундамент для будущих функций.

## 🚀 Установка

1. Установи расширение **[Tampermonkey](https://www.tampermonkey.net/)**
   (Chrome, Edge, Firefox, Opera).
2. Открой панель Tampermonkey → **«Создать новый скрипт»**.
3. Удали весь шаблонный код и вставь содержимое файла `howrse-manager.user.js`.
4. Нажми **Ctrl + S** (сохранить).
5. Зайди на сайт **www.lowadi.com** — в правом верхнем углу появится окно 🐴.

---

## 🧑‍💻 Как помочь с настройкой (важно!)

Чтобы скрипт научился **точно** читать твоих лошадей:

1. Открой карточку любой лошади.
2. В окне приложения зайди в раздел **🧑‍💻 Разработчик**.
3. Нажми **«🔄 Обновить анализ»**.
4. Посмотри, что нашлось (✅), а что нет (❌).
5. Нажми **«📋 Скопировать отчёт»** и пришли его разработчику —
   так парсер будет доработан под реальную вёрстку страницы.

## 🔒 Безопасность

Всё работает **локально** в твоём браузере. Никакие данные никуда не отправляются.
Настройки и лог хранятся только у тебя (в `localStorage`).

---

## 🏗️ Архитектура

Проект построен по модульному принципу, чтобы легко расти:

- `EventBus` — общение между частями через события
- `Storage` — сохранение данных
- `Logger` — журнал действий
- `SettingsManager` — настройки
- `StateManager` — текущее состояние приложения
- `RouteManager` — определение типа страницы
- `HorseParser` — чтение данных лошади
- `GameAdapter` / `LowadiAdapter` — слой под конкретную игру (легко добавить новые)
- `RunEngine` — «мозг» выполнения задач
- `UIManager` — весь интерфейс
