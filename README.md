# Howrse Manager

**Howrse Manager** — Tampermonkey-приложение для Ловади / Howrse.

Проект создаётся как будущий умный менеджер табуна: с плавающим интерфейсом, настройками, логом, анализом лошади и постепенным добавлением автоматизации.

## Текущая версия

`v0.4.1` — доработка миссий и кормления.

## Что есть сейчас

- плавающее окно приложения на `www.lowadi.com`;
- светлая, тёмная и автоматическая тема;
- сохранение настроек в браузере;
- кнопки `Старт`, `Пауза`, `Продолжить`, `Стоп`;
- кнопка `Стоп` сбрасывает счётчик обработанных лошадей;
- базовый лог событий;
- раздел разработчика;
- определение типа страницы;
- анализ текущей лошади:
  - имя;
  - энергия;
  - здоровье;
  - настроение;
  - возраст;
  - пол;
  - кнопка следующей лошади;
- гибридный табунный режим:
  - текущая лошадь;
  - базовый уход;
  - переход к следующей лошади;
  - ручной лимит или режим `Авто — до конца завода`;
- базовый уход:
  - чистка;
  - урок / миссия;
  - ласка;
  - вода;
  - корм;
  - сон;
- статистика обработанных лошадей и действий ухода.

## Изменения v0.4.1

- Миссия ищется в первую очередь по разделу `Миссия`, поэтому скрипт меньше зависит от конкретного текста кнопки.
- Добавлены варианты текста миссии:
  - `Транспортировать железо`;
  - `Транспортировать древесину`;
  - `Транспортировать песок`.
- Кормление стало двухшаговым:
  1. нажимается `Кормить`;
  2. в открывшемся окне читаются нормы вида `0 / 14` и `0 / 6`;
  3. выставляется остаток нормы для корма и овса;
  4. нажимается `Дать поесть`.

## Важно о v0.4.1

Механика кормления на Ловади может отличаться по DOM-разметке у разных аккаунтов/страниц. Если ползунок не выставится, пришлите скриншот раздела разработчика после нажатия `Обновить анализ` и, по возможности, HTML-фрагмент окна кормления.

Для первого теста рекомендуется поставить ручной лимит `1–3` лошади.

## Установка для теста

1. Установить расширение Tampermonkey.
2. Открыть файл [`howrse-manager.user.js`](./howrse-manager.user.js).
3. Скопировать содержимое файла в новый userscript Tampermonkey или обновить уже установленный.
4. Сохранить скрипт.
5. Открыть `https://www.lowadi.com/`.
